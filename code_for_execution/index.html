Minesweeper Cricket        
This is a very interesting game!!!!!! Play right now!!!!



The rules of the game are very simple...........

Single Player:
1) You have to randomly select a box in the grid.
2) If the box contains a fielder, then ""GAME OVER"".
3) If it does not contain fielder, then you get the points mentioned in the box.
4) You can get points in the range 1 to 6
5) The number of fielders in the game are fixed and equal to 11.
6) The game will continue till either you have covered all the possible boxes for runs or you have met a fielder.

Multiplayer:
1) The rules of Single Player game work in this mode as well. 
2) In this case, the player who scores more runs wins.
3) In case a player encounters a fielder, he is out of the game and the second player plays the rest of the game.

Special Attributes:
There are two special attributes offered in this game
1) """Multiplier""" `X2` This attribute multiplies your points by 2 every time you achieve it.
2) """Protection""" `Pr` This attribute protects you from 1 of the fielders. The more you get these, the protected you are.
3) The total number of both the attributes are fixed in a game and equal to the grid size you choose.
